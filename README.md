# Convertisseur d'Unites

Outil en ligne de commande pour convertir entre differentes unites.

## Fonctionnalites

- Kilometres ↔ Miles
- Degres Celsius ↔ Fahrenheit
- Euros ↔ Dollars
- Kilogrammes ↔ Livres (lb)
- Menu interactif avec validation des entrees

## Compilation
```bash
gcc main.c -o convertisseur
```

## Utilisation
```bash
./convertisseur
```

## Exemple
```
=== Convertisseur ===
1. Kilometres <--> Miles
2. Degres Celsius <--> Fahrenheit
3. Euros <--> Dollars
4. Kilogrammes <--> Livre (lb)
5. Quitter

Choisir le numero d'une operation : 1
1. Kilometre en Miles
2. Miles en Kilometres
Choisir : 1
Entrer le nombre de kilometres : 100
100.00 kilometres est egal a 62.14 miles
```

## Competences

- switch/case
- Conversions mathematiques
- Validation des entrees
- Menu interactif
